# Milestone 2

**Hi all, welcome to UBC STAT545 mini data analysis project - Milestone 2!**

## Introduction

This is the [Milestone 2](https://stat545.stat.ubc.ca/mini-project/mini-project-2/) of the mini data analysis project.

## Learning Objectives

- Become familiar with manipulating and summarizing your data in tibbles using ```dplyr``` and ```tidyr```, with a research question in mind.

- Understand what tidy data is, and how to create it.

- Generate a reproducible and clear report using R Markdown.

- Gain a greater understanding of how to use R to answer research questions about your data.

## List of Files

- ```mini-data-analysis-2.Rmd```: source code of data analysis in Milestone 2.

- ```mini-data-analysis-2.md```: Readme version of ```.Rmd``` files.

- ```mini-data-analysis-2_files```: folders containing all graphs generated from ```.Rmd``` files.

- ```new_vancouver_trees```: the new dataframe generated from the existing dataset ```vancouver_trees``` in Milestone 2.
